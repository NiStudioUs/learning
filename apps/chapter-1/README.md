# 🍔 KS Foodie - Online Food Ordering App

**KS Foodie** is a premium, responsive front-end web application designed to simulate a modern food ordering experience. It combines a rich "Zomato-Red" aesthetic with smooth interactions, glassmorphism effects, and a seamless "Infinite Scroll" food discovery engine.

## 🚀 Key Features

### 🌟 UX/UI
- **Modern Split-Screen Login:** A visually stunning entry point with high-quality imagery.
- **Glassmorphism:** Elegant semi-transparent elements for a modern feel.
- **Smooth Animations:** Integrated micro-interactions for a polished experience.
- **Fully Responsive:** Flawless layout on Mobile, Tablet, and Desktop.

### 🛠 Core Functionality
- **Dynamic Food Catalog:** Powered by [TheMealDB API](https://www.themealdb.com/api.php).
- **Infinite Scroll:** Endless food browsing without page reloads.
- **Smart Pricing:** Auto-generated consistent pricing for demo purposes.
- **Cart Management:** LocalStorage-based cart with real-time updates.
- **Mock Admin:** Simulated admin panel for menu management.
